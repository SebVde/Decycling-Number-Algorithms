# Algorithmes pour le decycling number de graphes simples non orientés

Vous trouverez dans ce projet différents algorithmes exacts et approchés permettant de calculer le decycling number dans les graphes simples non orientés. Il est également possible de lancer un benchmark sur ces algorithmes afin de tester leur efficacité et leur rapidité d'exécution sur différents jeux de graphes.

---

## Prérequis

- Python 3.12 ou supérieur

Afin d'installer les dépendances nécessaires au bon fonctionnement des méthodes et du benchmark, veuillez lancer la commande :

```bash
pip install -r requirements.txt
```
